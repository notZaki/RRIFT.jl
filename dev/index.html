Introduction

This is a Julia module which implements the reference region and input function tail (RRIFT) method described in:

Zaki Ahmed & Ives R. Levesque. Pharmacokinetic modeling of dynamic contrast‐enhanced MRI using a reference region and input function tail. Magn Reson Med. 2020; 83: 286– 298. https://doi.org/10.1002/mrm.27913

Installation

The module can be installed by

julia> ]add https://github.com/notZaki/RRIFT.jl.git#master

Once installed it can be used by

julia> using RRIFT
